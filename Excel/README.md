# Milestone-1
# Stock Returns Analysis: OpenDoor vs. Toyota

## Project Overview
This project analyzes weekly stock returns of OpenDoor and Toyota over 250 observations. Using descriptive statistics and regression analysis, the study evaluates volatility, risk, and the relationship between the two stocks. The analysis provides insights into stock behavior and investment decision-making.

## Key Quantitative and Digital Skills Demonstrated
- **Descriptive Statistics:** Calculated mean, range, quartiles (Q1, Q3), interquartile range (IQR), mean absolute deviation, variance, and standard deviation to summarize financial data.
- **Regression Analysis:** Conducted linear regression to examine relationships between stock returns, interpreted coefficients, R², and statistical significance.
- **Data Interpretation:** Analyzed trends, risk, and volatility, drawing actionable insights for investment strategies.
- **Data Visualization:** Used scatterplots and charts to illustrate patterns and regression relationships.
- **Digital Tools:** Utilized spreadsheet software (Excel) to perform statistical calculations and generate visualizations efficiently.

## Outcome
The project demonstrates the ability to combine quantitative analysis with digital tools to evaluate financial data, assess risk, and communicate findings clearly.
